Error Charts > Error Line > Creating a Simple Error Line Chart

Here, we' ll create a simple error line chart to show the change in temperature in different US cities on a particular date in two successive years.

The final chart would look as under:

The data for the above chart would look like:

  • XML
  • JSON
<chart caption="Temperatures Analysis" yAxisName="Temperature" numberSuffix="C" showValues='0' halfErrorBar="0" errorBarColor='666666'>
  
  <categories>
    <category label="Houston"/>
    <category label="San Francisco"/>
    <category label="New York"/>
    <category label="Chicago"/>
    <category label="Detroit"/>
  </categories>

  <dataset seriesName='1st Mar 2007'>
    <set value="-3.5" errorValue="3"/>
    <set value="6.6" errorValue="2"/>
    <set value="-3.5" errorValue="4.5"/>
    <set value="8.5" errorValue="2.5"/>
    <set value="5" errorValue="3.7"/>
  </dataset>

  <dataset seriesName='1st Mar 2008'>
    <set value="4" errorValue="3"/>
    <set value="10" errorValue="2"/>
    <set value="4" errorValue="4.5"/>
    <set value="14" errorValue="2.5"/>
    <set value="14.5" errorValue="3.7"/>
  </dataset>

</chart>

In the data above, we have done the following:

  • Created the <chart> element and defined caption, y axis name, number suffix and disabled the display of datavalues in the chart. Also, in order to show full error bars, we have set halfErrorBar='0' as by default, only positive deviation is displayed.
  • Next, there lies the errorBarColor attribute which allows you to set a color for the error bars. Various other visual and functional aspects of error bars have been discussed in the Configuration page. Moreover, you can specify innumerable properties for the <chart> element which will let you control a myriad of functionalities in your chart. The entire list of attributes is present in the Chart XML Sheet.
  • Moving on, we have <categories> element, in which you can define a custom label for each <category> element.
  • Thereafter, we added two <dataset> elements and provided data for the respective series using <set> elements. In every <set> element, along with the datavalues we have also defined the range of possible errors as the value of errorValue attribute.
