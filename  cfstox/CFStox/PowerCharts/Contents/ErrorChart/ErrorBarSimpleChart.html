Error Charts > Error Bar > Creating a Simple Error Bar Chart

Let's create a simple error bar chart which compares the life expectancies of common household appliances both at home and in office.

The chart would look as under:

The XML/JSON for the above chart can be listed as under:

  • XML
  • JSON
<chart caption='Life Span chart of Appliances' yaxisname='Life Span (year)' showValues='0' halfErrorBar='0' errorBarColor='666666'>
  
 <categories>
      <category label='Microwave oven' />
      <category label='Air Conditioner' />
      <category label='Refrigerator' />
      <category label='Coffee Machine' />
      <category label='Dish Washer' />
   </categories>

   <dataset seriesName='At Home' >
      <set value='9' errorValue='2' />
      <set value='11' errorValue='1.5' />
      <set value='16' errorValue='3' />
      <set value='10' errorValue='1.8' />
      <set value='8' errorValue='1.2' />
   </dataset>

   <dataset seriesName='In Office' >
      <set value='5' errorValue='1' />
      <set value='6' errorValue='0.5' />
      <set value='13' errorValue='2' />
      <set value='7' errorValue='0.8' />
      <set value='5' errorValue='1' />
   </dataset>

</chart>

In the data above, we have:

  • Created the <chart> element and defined caption, Y axis name and disabled the display of datavalues in the chart. Also, in order to show full error bars, we have set halfErrorBar='0' as by default, only positive deviation is displayed.
  • Next, the errorBarColor attribute is used to set a color for the error bars. Various other visual and functional aspects of error bars have been used which are discussed in the Configuration page. Moreover, you can specify innumerable properties for the <chart> element which will let you control various functionalities of your chart. The entire list of attributes is present in the Chart XML Sheet.
  • Moving on, we have <categories> element, which contain definitions of X axis label for each <category> element.
  • Thereafter, we added two <dataset> elements and provided data for the respective series using <set> elements. In every <set> element, along with the datavalues we have also defined the range of possible errors as the value of errorValue attribute.
