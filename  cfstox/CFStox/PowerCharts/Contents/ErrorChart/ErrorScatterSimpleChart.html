Error Charts > Error Scatter > Creating a Simple Error Scatter Chart

Let's create a simple error scatter chart that will display the comparison in performance of two fictional servers (Server 1 and Server 2). The chart will plot individual server load with respect to response time against x-axis and y-axis respectively as shown below.

The data for the above chart can be listed as under:

  • XML
  • JSON
<chart caption='Server Performance' xAxisName='Server Load (TPS)' yAxisName='Response Time (sec)' useHorizontalErrorBar='1' 
       halfHorizontalErrorBar='0' halfVerticalErrorBar='0' horizontalErrorBarColor='000000' verticalErrorBarColor='000000' >
  
  <categories>
    <category label='20' x='20' />
    <category label='40' x='40' />
    <category label='60' x='60' />
    <category label='80' x='80' />
    <category label='100' x='100' />
  </categories>
  
  <dataset seriesName='Server 1' color='009900' anchorRadius='5' anchorBgColor='7ED37E' >
    <set y='11' x='10' errorValue='3'  />
    <set y='9' x='35' errorValue='2'  />
    <set y='4' x='50' errorValue='2.5'  />
    <set y='14' x='70' errorValue='3' />
    <set y='14' x='50' errorValue='2'  />
    <set y='3' x='65' errorValue='2.5'  />
    <set y='17' x='85' errorValue='2' />
    <set y='15' x='15' errorValue='2'  />
  </dataset>

  <dataset seriesName='Server 2' color='0000FF' anchorRadius='4' anchorBgColor='85A5E9' >
    <set y='14' x='23' errorValue='2'/>
    <set y='15' x='33' errorValue='3.5'/>

    <set y='7' x='30' errorValue='1.5'/>
    <set y='11' x='55' errorValue='2'/>
  </dataset>
  
</chart>

In the data above, we have:

  • Created the <chart> element which is the starting element of every chart.
  • Specified caption, x-axis name and y-axis name. Also, we have instructed the chart to show horizontal error bars by setting the useHorizontalErrorBar attribute to 1. The chart, by default displays the half vertical error bars.
  • In order to show full horizontal and vertical error bars, we have deactivated the halfHorizontalErrorBar and halfVerticalErrorBar attributes respectively as by default, only positive deviation is displayed.
  • Next, there lies the horizontalErrorBarColor and verticalErrorBarColor attributes which allow you to set a color for the corresponding error bars. Various other visual and functional aspects of error bars have been discussed in the Configuration page. Moreover, you can specify innumerable properties for the <chart> element which will let you control a myriad of functionalities in your chart. The entire list of attributes is present in the Chart XML Sheet.
  • Moving on, we have <categories> element which represents a custom label for each <category> element.
  • Thereafter, we add two <dataset> elements and defined series name, color, anchor radius and anchor background color for both the series.
  • Finally, we provide data for the respective series using <set> elements. In scatter charts every <set> element can consist of multiple numeric parameters. Here, in this chart, the position of a data point is defined in form of two numeric parameters namely x and y. Additionally, in every <set> element, along with the datavalues we have conjointly defined the range of possible errors (horizontal and vertical) as the value of errorValue attribute.

You can also explicitly specify the amount of horizontal and vertical errors with the help of horizontalErrorValue and verticalErrorValue attributes respectively. Hence, a <set> element would look something as under:

<set x='10' y='11' horizontalErrorValue='3' verticalErrorValue='4'/>
