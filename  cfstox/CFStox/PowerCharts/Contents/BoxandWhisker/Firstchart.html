Box and Whisker > Creating a simple chart

To begin with, let's create a chart showing the number of visits to a website for the months of January and February in three different years. The chart will look as under :

And the data for same can be listed as under:
  • XML
  • JSON
<chart caption='Visits to a Website' subCaption='In three years'>

<categories>
  <category label='2008' />
  <category label='2009' />
  <category label='2010' />
</categories>

<dataset seriesName="January" lowerBoxColor='009FAA' upperboxColor='D4DFFF'>
    <set value='160,133,35,67,89,107,59' />
    <set value='139,67,75,114,39,67,191' />
    <set value='200,107,197,289,53,26,97' />
</dataset>

<dataset seriesName="February" lowerBoxColor='2A5F55' upperboxColor='D47F00'>
    <set value='160,303,125,137,169,107,89' />
    <set value='139,167,255,124,99,187,151' />
    <set value='60,137,117,219,153,126,277' />
</dataset>

</chart>

        },

        {

        }

In the above XML we have done the following:

  • Created the <chart> element with two attributes to define the caption and the sub-caption of the chart
  • Defined years as x-axis labels using <category> elements
  • Created two <dataset> elements (January and February) to provide data
  • Provided website visit data using the value attribute in each of the <set> elements
  • Set the colors for the upper and lower boxes of the dataplots by the lowerBoxcolor and upperBoxColor attributes in <dataset> elements

    • When the chart loads the data, it shows box and whisker plots. The Minimum value, maximum value and the median value are displayed whereas, the upper and lower quartile values are hidden by default.
