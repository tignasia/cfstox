Box and Whisker chart > How it works?

'Five-number summary' principle is used to render PowerCharts' Box and whisker plots. This principle helps to provide a statistical summary for a given set of numbers. It gives information about the range (minimum and maximum numbers), the center (median) and the spread (upper and lower quartiles) for the set of values provided. Note: There is another principle named as 'Seven-number summary' which is not used in the current implementation.

To execute the five-number summary we have used the Quartiles method. Using this method, a set of numbers are divided into four equal parts by three quartiles {Q1 (lower quartile), Q2 (median), and Q3 (upper quartile)}.

Note: There are two other methods as Deciles and Percentiles to execute the five-number summary which are not used in this implementation.

The formulas used in this chart are:

  • Median (Q2): N+1/2 (where N=total number of elements in a set of data)
  • Lower Quartile (Q1): N+1/2 (where N=total number of elements in a set of data)
  • Upper Quartile (Q3): 3(N+1)/2 (where N=total number of elements in set of data)
  • Minimum number: Displays the smallest number in a set of data
  • Maximum number: Displays the largest number in a set of data
  • Mean (M): ∑Xi /N (where N=total number of elements in a set of data, x=values present in the set of data, and i=position of x in a set of data)
  • MD (Mean deviation): ∑f |Xi - mean|/N (where N=total number of elements in a set of data, f=frequency, x=values present in the set of data, i=position of x in the set of data)
  • SD (Standard deviation): √ (∑ (Xi - mean) ²/N) (where N=total number of elements in a set of data, x=values present in the set of data and i=position of x in the set of data)
  • QD (Quartile Deviation): Q3-Q1

Note: All the above calculations are made after sorting the set of data in ascending order.
Providing Data

Data is provided In Box and whisker chart in the format similar to other multi series charts. The only difference is, instead of providing a single value we provide data as a set of numbers.

By default ‘,’ (comma) is used to separate the numbers. The data separator is customizable. You can set the symbol as per your requirement by using the attribute dataSeparator in the <chart> element. For example, you can set dataSeparator = ';' and then use the same symbol in the <set> element to separate the numbers (35; 57; 89; -14).

The data for same can be listed as under:
  • XML
  • JSON
<chart caption='Visits to a Website' subCaption='In three years' dataSeparator=';'>
...
<dataset seriesName="January" lowerBoxColor='55236E' upperboxColor='CBBFF9'>
