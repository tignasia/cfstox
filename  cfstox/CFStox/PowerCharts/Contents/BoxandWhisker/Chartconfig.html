Box and Whisker chart > Configuring five summary numbers
PowerCharts allows you to customize all the 5 summary numbers and their deviations according to your requirement. In this page we will discuss the following:

  • Show or hide the five summary numbers
  • Customize the median
  • Customize the quartiles and the quartile boxes
  • Customize the whiskers or the upper and lower limit
Show or Hide the five summary numbers

The six attributes which are used to control the display of the five-summary numbers are:

Attribute name for <chart> and <dataset> elements Attribute name for <set> element Type Range Default Value Description
showValues showValue Boolean 0/1 1 Sets whether to show or hide the five summary numbers. If this attribute is set to '0', none of the summary numbers will be displayed. When this is set off the individual settings through showMinValues, showMaxValues, showMedianValues, showQ1Values, showQ3Values attributes will not work.
showMinValues showMinValue Boolean 0/1 1 Sets whether to display or hide the minimum number or the lower limit.
showMaxValues showMaxValue Boolean 0/1 1 Sets whether to display or hide the maximum number or the lower limit.
showMedianValues showMedianValue Boolean 0/1 1 Sets whether to display or hide the median.
showQ1Values showQ1Value Boolean 0/1 0 Sets whether to display or hide the lower quartile.
showQ3Values showQ3Value Boolean 0/1 0 Sets whether to display or hide upper quartile value.

Note that:
  • Attributes used in the <chart> element are applicable globally to all the dataplots of the chart
  • The setting of the attributes declared in the <dataset> element overrides the settings declared in the <chart> element
  • The setting of the attributes declared in the <set> element overrides the settings declared in the other two elements

Show all five-summary numbers

Let's see how all the five-summary numbers displayed in a chart. The chart will look like as under:

The settings for the above chart is given below through showQ1Values and showQ3Values attributes:
  • XML
  • JSON
<chart caption='Visits to a Website' subCaption='In three years' showQ1Values='1' showQ3Values='1'>
...
</chart>
{
  "chart":{
    "caption":"Visits to a Website",
    "subcaption":"In three years",
    "showq1values":"1",
    "showq3values":"1"
  }
}
In the above data, within the <chart> element we have set showQ1Values and showQ3Values to '1'. This allows the chart to display the lower and the upper quartile values along with the minimum, maximum and the median values which are displayed by default.

To hide the five summary numbers selectively you need to individually set the value of the respective attribute to '0' which you may not want to display in the chart. Let's create a chart displaying only the maximum and minimum values. The chart will look like as under:

The setting required in the data is as follows:
  • XML
  • JSON
<chart caption='Visits to a Website' subCaption='In three years' 
showQ1Values='0' showQ3Values='0' showMedianValues='0' showMinValues='1' showMaxValues='1'>
...
</chart>
{
  "chart":{
    "caption":"Visits to a Website",
    "subcaption":"In three years",
    "showq1values":"0",
    "showq3values":"0",
    "showmedianvalues":"0",
    "showminvalues":"1",
    "showmaxvalues":"1"
  }
}
In the above data,
  • The showValues attribute is set to '1' to primarily display all the values in the chart
  • The attributes showMedianValues, showQ1Values and showQ3Values are set to '0' in order to hide median, Q1 and Q3 values
  • The attributes showMinValues and showMaxValues are set to '1', hence, displaying only these values

Customizing the Median (Q2)

The "Median" for a set of numbers is represented by a line at the center of the box plot. The median acts as the separator between the upper and lower quartile boxes.

The attributes used to customize the median are given below:

Attribute Name Type Range Description
medianColor Hex code   Sets the color of the line which represents the median.
medianThickness Number   Sets the thickness of the median line.
medianAlpha Number 0-100 Sets the transparency of the median line.

Note: All the three attributes can be used in the <chart>, <dataset> and <set> elements.

Customizing the upper and lower quartiles (Q1 and Q3)

The quartiles of a set of data are spread as two boxes separated by the median. The roof of the upper quartile box represents the exact value of the upper quartile (Q3) . The base of the lower quartile box represents the exact value of the lower quartile (Q1).

The attributes used to customize the quartile points and the boxes are discussed in the table below:

Attribute Name Type Range Description
upperQuartileColor Hex code   Sets the color of the line that represents the exact value of the upper quartile. This line is the roof of the upper quartile box.
UpperQuartileThickness Number   Sets the thickness of the upper quartile line.
UpperQuartileAlpha Number 0-100 Sets the transparency of the upper quartile line.
lowerQuartileColor Hex code   Sets the color of the line that represents the exact value of the lower quartile value. This line is the base of the lower quartile box.
lowerQuartileThickness Number   Sets the thickness of the lower quartile line.
lowerQuartileAlpha Number 0-100 Sets the transparency of the lower quartile line.
upperBoxColor Hex code   Sets the color of the upper quartile box.
upperBoxAlpha Number 0-100 Sets the transparency of the upper quartile box.
lowerBoxColor Hex code   Sets the color of the lower quartile box.
lowerBoxAlpha Number 0-100 Sets the transparency of the lower quartile box.
upperBoxBorderColor Hex code   Sets the color of the two sides of the upper quartile box.
upperBoxBorderThickness Number   Sets the thickness of the two sides of the upper quartile box.
upperBoxBorderAlpha Number 0-100 Sets the transparency of the two sides of the upper quartile box.
lowerBoxBorderColor Hex code   Sets the color of the two sides of the lower quartile box.
lowerBoxBorderAlpha Number 0-100 Sets the transparency of the two sides of the lower quartile box.

Customizing the minimum and maximum numbers (lower and upper whiskers)

The chart displays the biggest number of a set of values as the upper whisker and the smallest number of the set as the lower whisker.

The attributes used to customize the whiskers are discussed in the table below:

Attribute Name Type Range Description
upperWhiskerColor Hex code   Sets the color of the upper whisker of the data plot.
upperWhiskerThickness Number   Sets the thickness of the upper whisker of the data plot.
upperWhiskerAlpha Number 0-100 Sets the transparency of the upper whisker of the data plot.
lowerWhiskerColor Hex code   Sets the color of the lower whisker of the data plot.
lowerWhiskerThickness Number   Sets the thickness of the lower whisker of the data plot.
lowerWhiskerAlpha Number 0-100 Sets the transparency of the lower whisker of the data plot.
whiskersLimitWidthRatio number   Sets the width of the whiskers in percentage with respect to the width of the box plot.
