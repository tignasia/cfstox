Box and Whisker chart > Mean deviation

Mean deviation is the average of distances between each of the values and the mean. By default, the mean deviation is represented by a triangle icon in the data plot. To display the mean deviation set the showMD attribute to ‘1’.

Note: The attribute can be used in the <chart>, <dataset> and <set> elements.

A chart displaying the mean deviation is shown below:

In the above screenshot, the chart is displaying the mean deviation icons of each and every data plot. This icon is customizable about which we will discuss later.

The data for the above chart is given below:
  • XML
  • JSON
<chart caption='Visits to a Website' subCaption='In three years' showMD='1' MDIconSides='8' showValues='0'>
...
</chart>
{
  "chart":{
    "caption":"Visits to a Website",
    "subcaption":"In three years",
    "showmd":"1",
    "mdiconsides":"8",
    "showvalues":"0"
  }
  ...
}


In the above XML, the showMD attribute is set to '1' in the <chart> element. So, all the data plots are displaying the mean deviation icon. The MDIconSides is set to '8' to display the icon as a circle.

You may want to display the mean deviation icon of one dataset and hide the other. To do this use the showMD attribute in the <dataset> element. A sample data is given below:

  • JSON

...

The attributes used to customize the mean deviation icon are given below:
Attribute Name Type Range Description
MDIconShape String   Specifies the shape of the icon. There are two options polygon and spoke.
MDIconRadius Number   Specifies the radius of the mean deviation icon.
MDIconSides Number   Specifies the number of sides for the icon. If the MDIconShape is set as polygon, then the shape of the icon will depend on the number of sides. For example, 3 sides will display a triangle, 8 will display a circle.
MDIconColor Hex code   Sets the color of the mean deviation icon.
MDIconAlpha Number 0-100 Sets the transparency of the mean deviation icon.

Mean deviation Connectors

For multiple dataplots, you can draw a line to connect the mean deviation icons of each dataplot. The attribute drawMDConnector is set to '1' to connect all the icons of the dataplots. This attribute can be used in <chart> and <dataset> elements. The color of the connecting line will be same as the color of the mean deviation icon.

Connect Null data

If you set the showMD attribute in any of the <set> element to ‘0’, the chart will not display the mean deviation icon of that particular set of data. In this case, the missing mean deviation is considered as null data. Consequently, the mean deviation connectors will skip this null data.

An example of this situation is shown below:



The XML data is given below:

  • XML
  • JSON
<chart caption='Visits to a Website' subCaption='In three years' drawMeanConnector='1'>
...
<dataset seriesName="January" lowerBoxColor='2A5F55' upperboxColor='D47F00' MDIconsides='5' showMD='1' mediancolor='FF0000'>
    <set value='60,133,35,67,89,137,259' showMD='1'/>
    <set value='139,67,75,114,39,67,191' showMD='0'/>
    <set value='200,107,197,239,53,26,97' showMD='1'/>
	<set value='210,117,187,219,73,96,171' showMD='1'/>
	<set value='230,107,157,259,93,116,191' showMD='1'/>
</dataset>
</chart>
{
  "chart":{
    "caption":"Visits to a Website",
    "subcaption":"In three years",
    "drawmeanconnector":"1"
  },
  ...

      "upperboxcolor":"D47F00",
      "mdiconsides":"5",
      "showmd":"1",

In the above XML:

  • The showMD attribute is set to '0' in the third <set> element to hide the mean deviation icon.
  • The drawMDConnector is set to '1' to connect the icons

The mean deviation connector in the above chart is drawn as a discontinuous line.

You can opt to ignore the null data and draw a continuous connector. To achieve this set connectNullData = '1'. In our above example, the chart will draw the line connecting the mean deviation icons of the first, second and the fourth, fifth dataplots skipping the third dataplot. This attribute can only be used in the <chart> element.
