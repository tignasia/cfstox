In the above XML :

  • The <set> elements are used within the <dataset> element to provide data for the rows and columns of the chart
  • There are two attributes, rowId and columnId, which are used to identify a cell of the table in which the data values are plotted
  • In the <colorRange> element the lower limit for the numeric range is defined using the minValue attribute
  • The starting color for the range of colors is also set in the <colorRange> element using the code attribute
  • The gradient attribute is set to '1' to show gradient legend. If this attribute is set to 0 then the chart will display icon legends. For more details see the Icon legend page
  • In each <color> element the upper limit of the numeric range and the colors associated to that range is declared
  • The upper limit is set using the maxValue attribute and the colors associated are set using the code attribute

Primarily there can be two types of heat map chart. They are:

  • Numeric based heat map chart
  • Category (non-numeric) based heat map chart
