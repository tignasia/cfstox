Multi-level Pie Chart > Creating a Simple Chart

Since multi-level pie charts are essentially used in displaying hierarchical data, here we'll create a simple chart to show you the social network structure of a person. The segments in the chart have been categorized according to their status.

The chart can look something as under:

The XML/JSON for the above chart is listed below:

  • XML
  • JSON
<chart caption='Tom&apos;s Friend List' plotfillAlpha='30' >

In the data above, we have:

  • Defined caption and alpha of the plot
  • Added the root <category> element and all the other sub <category> elements are defined as per the hierarchical tree structure
