Overview

PowerCharts v3 helps you plot domain-specific charts that are highly interactive. These charts can be used for a variety of purposes like simulations, visual data selection, scientific plotting, financial analysis etc. PowerCharts is one of the 4 products of FusionCharts v3 suite, the others being FusionCharts, FusionMaps and FusionWidgets.

Since PowerCharts is based on FusionCharts v3 framework, it utilizes most of the features provided by the parent v3 suite itself. Like FusionCharts v3, PowerCharts v3 leverages Flash and JavaScript (HTML5) to create stunning data visualizations unmatched by conventional server-side components. As it works with both XML and JSON data, it can be integrated with any server-side technology (ASP, ASP.NET, PHP, JSP, ColdFusion, Ruby on Rails etc.) and database.

Be it a simple spline or inverse chart, radar chart, advanced drag-able, predictive & selectable charts or specialized stock charts, network/flow charts, PowerCharts covers all your advanced charting requirements. All the charts support interactive options like tooltips, drill-down, exporting as image/PDF/CSV and JavaScript integration.

This documentation focuses more on PowerCharts side of story rather than going back to basics of FusionCharts. For FusionCharts basics, you can refer to FusionCharts v3 Documentation or see "FusionCharts v3 Basics" section reproduced in this documentation. In the introductory part (FusionCharts v3 Basics) of this documentation, we've used the Column 2D, Column 3D and Pie 3D charts from FusionCharts v3 suite to explain generic features provided by FusionCharts v3 framework. These features can be applied to each of the PowerCharts charts as well.

The Column 2D, Column 3D and Pie 3D charts discussed in FusionCharts v3 Basics section are NOT part of PowerCharts v3. But these being the simplest charts in the entire FusionCharts suite, things will be much easier for you to grasp when explained in perspective of these charts. Once you're a pro with the basics, you can easily apply the same concepts to the charts you desire to use.

Also, throughout this documentation, we'll synonymously use the term FusionCharts v3 and PowerCharts v3.

Advantages of using PowerCharts
Works with all web technologies and databases
 

PowerCharts works with ASP, ASP.NET, PHP, JSP, ColdFusion, Ruby on Rails, JavaScript or even simple HTML pages. It connects to any database (MS SQL, MySQL, Oracle, PostgreSQL) and web feeds to enable dynamic charting.

 Unique chart types available
 

PowerCharts offers a unique set of charts, that you might not have seen on the web before. These charts are generic chart types that can be used for a variety of purposes. There are charts like Heat map chart, Box and Whisker chart, Waterfall/Cascade and many more.

Stunning looks & complete reporting
 

PowerCharts not only gives your data a pretty face, it also adds smart reporting features like tooltips, drill-down, chart export, interactive data-selection, interactive data-prediction. Some charts also allow visual editing of data using simple drag-n-drop.
Animated and interactive Charts that "wow" your audience
 

Using PowerCharts, you can quickly and easily render animated charts that have a lot of interactive options for the end users. Different chart types support different forms of animation and interactivity, thereby providing a unique experience to the end users.
Accepts both XML and JSON data
 

PowerCharts accepts both XML and JSON data, enabling you to use data format of your choice. Using XML you can connect it to your enterprise applications very easily and with JSON to your JavaScript frameworks.
Reduces load on your servers
 

In traditional image based charting systems, the charts are generated as image at the server side. So, for each chart that you need to serve to a user, you'll have to build complex images on the server and then stream it to the client. When the need of hour is high, this can be an expensive resource on the server, as image creation takes a toll on the server.

At this juncture, PowerCharts brings you great relief - all the charts are rendered at client side using either the widely installed Adobe Flash Platform or JavaScript. The server is just responsible for streaming the pre-built SWF/JS files and your XML/JSON data files to the end viewers. From there on, Flash Player/JavaScript engine takes the onus of rendering the charts. Also, the chart SWF/JS files can be cached so that you can just update the data, and not send chart SWF files every time.
Flash and JavaScript (HTML5) based charting
 

FusionCharts/PowerCharts renders charts using Flash and JavaScript (HTML5), giving you the best of both worlds. As such, it is the leading enterprise-grade charting component that works seamlessly on PCs, Macs, iPads, iPhones and most other mobile devices.
AJAX-enabled charts with extensive JavaScript API
 

All the charts in FusionCharts Suite are tightly integrated with JavaScript. This allows client-side chart or attribute update, listening to chart events, maintaining chart states or cloning it.
Unlimited levels of drill-down supported
 

FusionCharts/PowerCharts is a perfect fit for reports with multiple levels of drill-down. Individual data points on the chart can be configured to drill-down in same window, new window, pop-up window or frames. FusionCharts/PowerCharts also introduces LinkedCharts, an industry first, which lets you create unlimited drill down levels from a single data source; that too, without writing any additional code.
Exports to JPG, PNG, PDF and CSV
 

While charts are generated using Flash, they can also be exported to JPG, PNG or PDF. The data can also be copied from the chart as CSV. This can be done either through user interaction or even programmatically.
Industry-leading component trusted by Fortune 500 companies and the web’s most popular sites
 

With over 17,000 customers and 330,000 users, FusionCharts suite is the industry's leading RIA charting component. A majority of Fortune 500 companies prefer FusionCharts for their reporting needs. Popular websites that trust FusionCharts for their data visualization needs include LinkedIn, Weather.com, Federal IT Dashboard, AWeber, PollDaddy, StatCounter and Wufoo.

Commercially open source
 

You can purchase the entire source of FusionCharts as a part of the Enterprise Plus and OEM License.  This allows you to modify and recompile it as per your requirements. The code is written in ActionScript 2 and can be compiled using Flash IDE (8/CS3/CS4/CS5).
