Logarithmic Charts

The logarithmic column and line charts are very similar to the normal charts, but for the fact that these charts use a logarithmic axis instead of a normal linear axis.

Features:

  • You can select any positive base (apart from 1) for your logarithmic scale.
  • The logarithmic y-scale can be inversed to show charts like ranking charts etc.
  • Custom Selection of y-axis Lower and Upper Limit possible.
  • Custom Selection of minor divisional lines between any two major divisional lines.

Since these charts derive from the normal v3 charts, the feature set provided is same. Currently, there are 2 logarithmic charts present in the suite:

Let's throw some light upon the various features of a logarithmic chart.

Custom Selection of Base

The default base of the logarithmic chart is 10. But, you can set the base of the logarithmic scale as any positive number greater than 1. In order to change the default configuration, you need to specify the desired base as the value of base attribute. 

<chart base='5' ... >

Inverse Axis Support

Logarithmic charts allow you to plot the y-axis values inversely so that the y-axis upper limit appears at bottom of the chart canvas instead of top. To use inverted y-axis in your logarithmic charts, you need to set:

<chart invertYAxis='1' ... >

Defining Major and Minor Divisional Lines

Minor logarithmic divisional lines are displayed between consecutive divisional lines. By default, the chart automatically calculates the number of minor divisional lines depending on the logarithmic base defined in the chart. You can however, explicitly specify the number of minor divisional lines by declaring the numMinorDivLines attribute in the following way:

<chart numMinorDivLines='4' ... >
