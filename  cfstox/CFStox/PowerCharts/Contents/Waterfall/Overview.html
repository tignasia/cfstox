Waterfall (Cascade Chart) > Overview

A waterfall (cascade) chart is a special type of column chart, used to show how an initial value is increased and decreased by a series of intermediate values, leading to a final value.

Shown below is a simple waterfall chart developed using PowerCharts:

The data for this chart can be listed as under:

  • XML
  • JSON
<chart caption="Company Profit (in USD) - 2008" numberPrefix="$" connectorDashed="1" sumLabel="Total Profit">
    <set label="Product Revenue" value="420000" />
    <set label="Services Revenue" value="210000" />
    <set label="Total Revenue"  isSum="1"/>
    <set label="Fixed Costs" value="-170000" />
    <set label="Variable Costs" value="-136000" />
    <set label="Total Costs" isSum="1" cumulative="0"/>
</chart>
