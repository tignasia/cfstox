Limitations of JavaScript charts

As of now, the FusionCharts JavaScript renderer has the following limitations:

  • Trendlines values in all JavaScript charts do not get shown on the right-hand side of the chart
  • Dashed lines and borders are not supported in all JavaScript charts
  • Zero thickness value in Flash based charts produces a hair-line, while JavaScript based charts do not draw the line
  • plotFillAlpha='0' in Flash charts hides the plot border, it does not hide the plot boder in JavaScript charts. To hide the border, one needs to specify plotBorderAlpha='0' as well
  • Interactivity (Drag or Select) is not supported in Drag Node, Drag Column, Drag Line, Drag Area and Select Scatter charts
  • Gradient rendering in JavaScript charts is not supported
  • Color palettes (1-5) in JavaScript charts are not supported
  • Server-side export (as image/PDF) of JavaScript charts is limited only to download
  • Client-side export and client-side batch export are not supported by JavaScript charts
  • JavaScript based Candlestick chart does not support export features
  • Anchors in Radar chart which fall behind another are not clickable (while using link feature or showing tooltips)
  • Long labels in Drag Node chart can not be wrapped
  • Divline related attributes in Multi-axis Line chart do not work if specified in <axis> element
  • The hierarchical hover effect does not work in Multi-Level Pie chart
  • In Waterfall chart the connector-lines start from center of column
