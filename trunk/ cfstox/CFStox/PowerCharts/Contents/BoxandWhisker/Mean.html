Box and Whisker chart > Mean

Mean is the mathematical average for a set of numbers. To display the mean set showMean='1'. By default, the mean is represented by a triangle icon in the data plot.

Note: The attribute can be used in the <chart>, <dataset> and <set> elements.

A chart displaying the mean is shown below:

In the above screenshot, the chart is displaying the mean values of each and every data plot. This icon is customizable about which we will discuss later.

The setting for the above chart is given below:
  • XML
  • JSON
<chart caption='Visits to a Website' subCaption='In three years' showMean='1' showValues='0'>
...
</chart>
{
  "chart":{
    "caption":"Visits to a Website",
    "subcaption":"In three years",
    "showmean":"1",
    "showvalues":"0"
	...
  }
 
}

In the above data, the showMean attribute is set to '1' in the <chart> element. So, all the data plots are displaying the mean icon.

Show/hide individual mean values

You may want to display the mean value of one dataset and hide the rest. To do this use the showMean attribute in the <dataset> element. A sample data is given below:
  • XML
  • JSON
<chart caption='Visits to a Website' subCaption='In three years' showValues='0'>
...
<dataset seriesName="January" lowerBoxColor='009FAA' upperboxColor='D4DFFF' mediancolor='FF0000' showMean='1'>
    <set value='60,133,35,67,89,137,259'/>
    <set value='139,67,75,114,39,67,191' />
    <set value='200,107,197,239,53,26,97' />
</dataset>

<dataset seriesName="February" lowerBoxColor='2A5F55' upperboxColor='D47F00'>
    <set value='160,303,125,137,169,107,39' />
    <set value='139,167,255,124,59,187,151' />
    <set value='60,137,157,209,153,126,277' />
</dataset>
</chart>

You can also configure the display the mean icon selectively for the set of data declared through the <set> elements. The sample data is given below:
  • XML
  • JSON
<chart caption='Visits to a Website' subCaption='In three years' showValues='0'>
...
<dataset seriesName="January" lowerBoxColor='009FAA' upperboxColor='D4DFFF' mediancolor='FF0000' >
    <set value='60,133,35,67,89,137,259' showMean='1'/> 
    <set value='139,67,75,114,39,67,191' />
    <set value='200,107,197,239,53,26,97' showMean='1'/>
</dataset>
</chart>

{
  "chart":{
    "caption":"Visits to a Website",
    "subcaption":"In three years",
    "showvalues":"0"
  },
  ...
  "dataset":[{
      "seriesname":"January",
      "lowerboxcolor":"009FAA",
      "upperboxcolor":"D4DFFF",
      "mediancolor":"FF0000",
      "data":[{
          "value":"60,133,35,67,89,137,259",
          "showmean":"1"
        },
        {
          "value":"139,67,75,114,39,67,191"
        },
        {
          "value":"200,107,197,239,53,26,97",
          "showmean":"1"
        }
      ]
    }
  ]
}

In the above data, we have set showMean = '1' in two of the <set> elements within the <dataset> element. So, the chart will display the mean icon only for the first and the third data plot. The chart will look like as under:
The attributes used to customize the mean icon are given below:
Attribute Name Type Range Description
meanIconShape String   Specifies the shape of the icon. There are two options - polygon and spoke.
meanIconRadius Number   Specifies the radius of the mean icon.
meanIconSides Number   Specifies the number of sides for the icon. If the meanIconShape is set as polygon, then the shape of the icon will depend on the number of sides. For example, 3 sides will display a triangle, 8 will display a circle.
meanIconColor Hex code   Sets the color of the mean icon.
meanIconAlpha Number 0-100 Sets the transparency of the mean icon.

Mean Connectors

For multiple dataplots, you can draw a line to connect the mean values of each dataplot. The attribute drawMeanConnector is set to '1' to connect all the mean icons of the dataplots. This attribute can be used in <chart> and <dataset> elements. The color of the connecting line will be same as the color of the mean icon.

Connect null data

If you set the showMean attribute in any of the <set> element to ‘0’, the chart will not display the mean icon of that particular set of data. In this case, the missing mean is considered as null data. Consequently, the mean connectors will skip this null data.

An example of this situation is shown below:



The data is given below:

  • XML
  • JSON
<chart caption='Visits to a Website' subCaption='In three years' showMean='1' drawMeanConnector='1'>
...
<dataset seriesName="January" lowerBoxColor='2A5F55' showMean='1' upperboxColor='D47F00' mediancolor='FF0000'>
    <set value='60,133,35,67,89,137,259' showMean='1'/>
	<set value='139,67,75,114,39,67,191' showMean='1'/>
	<set value='200,107,197,239,53,26,97' showMean='0' />
	<set value='210,117,187,219,73,96,171' showMean='1'/>
	<set value='230,107,157,259,93,116,191' showMean='1'/>
</dataset>
</chart>
{
  "chart":{
    "caption":"Visits to a Website",
    "subcaption":"In three years",
    "showmean":"1",
    "drawmeanconnector":"1"
  },
  ...
  "dataset":[{
      "seriesname":"January",
      "lowerboxcolor":"2A5F55",
      "showmean":"1",
      "upperboxcolor":"D47F00",
      "mediancolor":"FF0000",
      "data":[{
          "value":"60,133,35,67,89,137,259",
          "showmean":"1"
        },
        {
          "value":"139,67,75,114,39,67,191",
          "showmean":"1"
        },
        {
          "value":"200,107,197,239,53,26,97",
          "showmean":"0"
        },
        {
          "value":"210,117,187,219,73,96,171",
          "showmean":"1"
        },
        {
          "value":"230,107,157,259,93,116,191",
          "showmean":"1"
        }
      ]
    }
  ]
}

In the above XML:
  • The showMean attribute is set to '0' in the third <set> element to hide the mean icon
  • The drawMeanConnector is set to '1' to connect the mean icons

The mean connector in the above chart is drawn as a discontinuous line.

You can opt to ignore the null data and draw a continuous connector. To achieve this set connectNullData = '1'. In our above example, the chart will draw the line connecting the mean icons of the first, second and the fourth, fifth dataplots skipping the third dataplot. This attribute can only be used in the <chart> element.
